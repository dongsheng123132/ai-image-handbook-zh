## 一、AI 作图能干嘛

AI 作图就是**用大模型「凭一句话画图」**。不用会 PS、不用懂设计，把想要的画面用大白话说出来，几秒出图。常见玩法：

- **文生图**：「画一只戴墨镜的橘猫，坐在霓虹灯下，赛博朋克风，竖版」
- **改图 / 垫图**：丢一张图进去，「把背景换成纯白做电商主图」「按这张构图换成我的产品」
- **多图融合**：丢几张参考图，「把这个人物放进这个场景」
- **海报 / 配图 / 表情包 / 漫画**：促销海报、公众号头图、表情包、四格漫画

U-King 里这套的**默认模型是 OpenAI 的 `gpt-image-2`**——目前**改图、多图融合最强、出图最稳、国内最可靠**的图像模型。面板里还能一键切另外两个，按需要选：

| 模型 | 什么时候用 |
|---|---|
| `gpt-image-2`（默认） | **通用首选**：电商图、海报、改图、多图融合、出图带文字，都用它最稳 |
| `seedream-4-0`（字节·即梦） | **二次元 / 国风 / 潮流插画**，艺术风格更出彩 |
| `wanx2.1-t2i-turbo`（阿里·通义万相） | **纯国产、出图快、写实风稳**，电商写实主图可选 |

> 拿不准就用默认的 `gpt-image-2`，它的改图能力是另外两个比不了的。

> ⚠️ **国内自己用 OpenAI 作图的坑**：要 OpenAI 账号、要海外信用卡、要翻墙——九成新手卡在这。U-King 用**虾盘云**接好了国内直连，**这些全不用**，充值即用。

## 二、出第一张图（U-King 内置作图）

1. 打开 U-King，进 **AI 作图**面板。
2. 在提示词框输入要画的内容，例如：
   > 一只橘猫宇航员，漂浮在太空，电影感光影，星空背景
3. **模型**选 `gpt-image-2`（默认就是它）。
4. **尺寸**选 `1024x1024`（方图）。
5. 点出图，几秒后图就出来了。出的图自动存进作图历史（关了 App 也不丢），可另存为 PNG。

**改图**：上传一张参考图，提示词写「把背景换成星空」「按这张的风格画我的产品」，它在你给的图上改。要做**多图融合**就传多张参考图。

## 四、gpt-image-2 的尺寸规则（重点，避坑）

`gpt-image-2` **只认这几档尺寸**，填别的会报「尺寸不支持」：

| 选项 | 比例 | 适合 |
|---|---|---|
| `1024x1024` | 1:1 方图 | 电商主图、头像、表情包 |
| `1024x1536` | 2:3 竖版 | 海报、手机壁纸、小红书 |
| `1536x1024` | 3:2 横版 | 公众号头图、封面、Banner |
| `auto` | 自动 | 拿不准就用它，让模型自己定 |

> ⚠️ 老教程里的 `1792x1024` 是 **DALL·E-3** 的档，`gpt-image-2` 用会报错。要 600×900 这类**印刷竖版**：用 `1024x1536`（同为 2:3）出图后再缩放到目标像素。换 `seedream`/`wanx` 等模型时，按它各自文档填支持的尺寸；填错了，系统会在报错里告诉你可用档位。

## 五、出图提示词基础（说对了才出好图）

同样一句话，老手出大片、新手出鬼图，差在**怎么描述**。一条好提示词按「主体 + 细节 + 风格 + 质量 + 尺寸」拼：

| 成分 | 作用 | 举例 |
|---|---|---|
| **主体**（必填） | 画什么 | 一只橘猫、一瓶护肤品、一个穿汉服的女孩 |
| **细节 / 场景** | 在哪、什么状态 | 戴墨镜、坐在霓虹灯下、手里拿着花 |
| **风格** | 什么画风 | 赛博朋克、国风水墨、3D 卡通、写实摄影、扁平插画 |
| **质量词** | 提清晰度/质感 | 高清、电影感、光影细腻、超精细 |
| **尺寸** | 选对档位 | `1024x1024` 方图 / `1024x1536` 竖版 / `1536x1024` 横版 |

**拼起来**：`一瓶白色护肤品 + 放在大理石台面、旁边有绿植 + 写实摄影风格 + 光影细腻高清 + 1024x1024`。

> 💡 三个提速技巧：① 不会描述就**丢参考图**让它「照这张感觉画」；② 出得不对别重写，**在原图上接着改**一句句调（gpt-image-2 改图很强）；③ 好用的提示词**存成模板**反复用（红皮书有现成模板库）。

## 六、常见问题 TOP5

<details>
<summary><b>1. 报「尺寸不支持」出不了图？</b></summary>

你填了 `gpt-image-2` 不认的尺寸。改用 `1024x1024` / `1024x1536` / `1536x1024` / `auto` 之一（见第四节）。要其它像素先按这几档出，再缩放。
</details>

<details>
<summary><b>2. 提示「余额不足 / Invalid token」？</b></summary>

虾盘云没额度了。去 [u-claw.org.cn/recharge](https://u-claw.org.cn/recharge) 充值（¥1≈50万token），扫码即到账，回来重试。
</details>

<details>
<summary><b>3. 出来的图和我想的不一样 / 不像？</b></summary>

九成是提示词太笼统。按「主体+细节+风格+质量+尺寸」补细节；还不像就**丢一张参考图**让它照着画，比纯文字稳得多；或在出的图上接着说「把 ___ 改成 ___」逐步逼近。
</details>

<details>
<summary><b>4. 提示「内容不合规 / 被和谐」？</b></summary>

作图模型有内容审核，涉及真人脸、明星、敏感词、暴露内容会被拦。换个说法绕开（去掉具体人名、改「原创卡通形象」）。这是平台规则，不是你配错了。
</details>

<details>
<summary><b>5. 改图没效果 / 参考图没被用上？</b></summary>

确认你是**上传了参考图**再写改动要求（如「把背景换成星空」），而不是纯文字。`gpt-image-2` 的改图/融合很强，但要先给它图。多图融合就传多张。
</details>
